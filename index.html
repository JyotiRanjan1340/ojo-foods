🔥 Connect Firebase (Free)

Right now each phone has its own copy of the app. To make orders sync between all phones, connect a free Firebase database.

Step 1 → console.firebase.google.com
Step 2 → Create project (free)
Step 3 → Build → Realtime Database → Create
→ Start in Test Mode
Step 4 → ⚙️ Project Settings → Web app (</>)
Step 5 → Copy the firebaseConfig object
Step 6 → Open ojo-foods.html in a text editor
Step 7 → Find PASTE_YOUR_apiKey_HERE
→ Replace all 7 values with yours
Step 8 → Save & reopen the HTML file ✅
💡 Once done, every order placed by any customer will instantly appear on the admin screen — no refresh needed!
